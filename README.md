# COSMIC BROTH Replica

本项目是对 `https://www.cosmicbroth.com/` 公开页面的本地复刻。页面运行时、字体、图片、动画素材和公开 API 快照均已本地化。

## 启动

```powershell
npm start
```

然后访问 `http://127.0.0.1:4173`。

外部社交链接仍指向原页面使用的 Bilibili、微博和番茄小说地址；其余首页资源可从本地加载。
